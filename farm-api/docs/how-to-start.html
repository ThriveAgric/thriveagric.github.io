How to Start

It is quite easy to work with Thrive Agric's APIs. Just sign up to get your API key and then call any of the APIs you need. And remember to use your API key in every API call.

How to start in 3 simple steps

1. Sign up to get an API key. Once you're registered, we will send you a welcome email that contains your API key and additional information on how to get started with our APIs.

2. Get Verified. Contact support to activate your API key. Within a couple of hours, it will be activated and ready to use.

3. Start using API for free. Find the complete description of API calls with a list of parameters and examples of responses in API documentation. Please, remember to use your API key in each API call.

Example of using API key in API call

Description: Test the availabilty of API services.

We do not process API requests without the API key.

API Endpoint:

"https://api.sandboxthriveagric.live/api/v1/test";

Parameters:

 


fetch('https://api.sandboxthriveagric.live/api/v1/test', {
    method: 'GET',
    headers: {
        access_key: `${token}`,
        'Content-Type':'application/json'
    }
})
